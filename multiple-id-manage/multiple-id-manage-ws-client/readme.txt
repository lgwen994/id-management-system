●npmでライブラリを取得する手順
package.jsonを右クリックし、実行 -> npm Installを実行します。

●開発したJSのコンパイル方法
　以下の設定をします。
　http://10.45.239.23/redmine/projects/staveware-core-v5-3/wiki/JavaScript%E9%96%8B%E7%99%BA%E7%92%B0%E5%A2%83#Eclipse上での自動ビルドの設定
　上記の設定をすると、JSファイルに変更があると自動的にコンパイルします。
　開発初期など、JSファイルを保存するたびにコンパイルを実行されると不便な場合は、プロジェクト -> 自動的にビルドの
　チェックを外すと、コンパイルが実行されません。コンパイルしたいときにチェックを入れるとコンパイルが実行されます。
